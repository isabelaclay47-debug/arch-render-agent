建筑渲染智能体 · 内测版 使用说明
========================================

【运行前需要】
1. Windows 电脑。
2. 已安装 Google Chrome。
3. 一个 ChatGPT 账号（能正常打开 chatgpt.com，通常需要 VPN）——主流程靠它生图。
   若只想用「提示词助手」的本地识图，可以不用账号。
4. 已安装 Python 3.10 以上（安装时务必勾选 Add Python to PATH）。
   没装也没关系：双击启动时会自动打开下载页，装完再双击一次即可。

【怎么用】
1. 把这个文件夹解压到任意位置（别放 C 盘根目录或系统目录）。
2. 双击「双击启动.bat」。
   · 首次会自动装依赖（1-3 分钟），并弹出一个专用 Chrome；
   · 在那个 Chrome 里登录一次 chatgpt.com，登录后别关它。
3. 稍等几秒会自动打开 http://127.0.0.1:5001 ，在网页里操作即可。
4. 用完想彻底停止：双击「停止服务.bat」。

【想离线本地识图（可选）】
到网页「提示词助手」页，切到「本地大模型」，按提示点「一键准备本地识图」，
会下载并安装 Ollama + 一个识图模型（需你同意、较大，一次即可，之后离线复用）。

【更新】
连网时点网页右上角「检查更新」即可升级，不用重新下载整个包。

【出问题】
看文件夹里的 logs\app.log 和 logs\supervisor.log。
