# 底图标注工作室（PS-lite）— 设计

日期：2026-07-07
分支：`feat/prompt-helper`

## 目标

在**渲染前的底图阶段**提供一个"类 Photoshop"的标注/配景工作室，解决："我一开始只想改一部分、加点配景，而不是让 AI 把整张都改了"。

诚实边界：**不是**搬来 Photoshop 全部功能（那有几千个）。是在现有画板引擎上扩建一个**实用子集**：标注工具 + 配景摆放 + AI 局部生成。

## 用户已确认

- 配景三种方式全要（① AI 画 / ② 内置 CC0 素材库 / ③ 用户自带素材）。
- 底图阶段就能标注。
- 风格向 Photoshop 看齐（工具化、图层化的操作手感）。

## 分阶段交付

### 阶段 1（地基）— 底图标注工作室 + 扩充工具
- 原图旁加「✏️ 标注底图」按钮，打开工作室 modal，画布载入当前底图。
- 工具（在现有 手画/直线/框选/套索/涂抹/橡皮擦 上扩充）：
  **箭头、椭圆、文字**；加 **颜色选择、笔刷粗细**；撤销/清空。
- 「完成」：把标注**压平**到底图，作为新的底图（存进上传框）。
- 独立引擎 `static/annotate.js`，不动现有"生成图局部修改"画板（避免回归）。

### 阶段 2 — 配景素材（③ 自带 + ② 内置）
- 素材面板：分类（人/树/车/其它）。
- ③ 用户拖 PNG 进面板 → 存 `workspace/_stamps/`，可拖到画布摆放/缩放/旋转/删除。
- ② 内置 CC0 起始库：`fetch_assets.py` 联网拉一小批**免费可商用**透明 PNG 到 `static/stamps/`，
  许可与来源记录在 `static/stamps/CREDITS.md`，交付前请用户确认许可。
- 素材是**内容**：压平后保留在底图里（真的贴上去）。

### 阶段 3 — ① AI 画配景 + 管道打通
- 工具「AI 画这里」：圈一个区域 + 写"加一个走路的人"。
- 区分**标注=指令** vs **素材=内容**：
  - 指令类（箭头/文字/AI 圈）：不压进最终渲染，转成**给导演的文字说明 + 区域蒙版**；
  - 内容类（PNG 素材、画笔涂改）：压平进底图。
- 渲染时：干净底图 + 标注说明 + 区域，一起交给导演对话，AI 只在圈内加配景、锁住形体。

## 数据流（阶段 3 完成后）

```
底图 → 标注工作室
  ├─ PNG 素材/画笔  → 压平进底图（内容）
  └─ 箭头/文字/AI圈  → 文字说明 + 区域蒙版（指令）
        ↓
开始渲染：干净底图 + 说明 + 蒙版 → 导演对话 → 只在指定区做改动，锁形体
```

## 不做（YAGNI）

- 不做图层混合模式、滤镜、色阶曲线等 PS 高级功能。
- 不做多图层任意堆叠管理（只区分"底图 / 标注 / 素材"三层语义）。
- 不追求像素级抠图工具（配景靠现成 PNG 或 AI 生成）。

## 验收（分阶段）

1. 阶段1：能打开工作室、用箭头/文字/椭圆/画笔标注底图、改颜色粗细、撤销、完成后底图带标注。
2. 阶段2：能拖入自己的 PNG 摆放缩放；内置素材库能选用；许可有记录。
3. 阶段3：圈区域+写指令，渲染只在圈内加配景、不动其余；标注指令不残留在成图里。
